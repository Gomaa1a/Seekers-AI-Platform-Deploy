# Seekers AI - Multi-Tenant SaaS Platform with n8n Workflows
## Complete Technical Specification for Enterprise Implementation

---

## PROJECT OVERVIEW

Build a complete multi-tenant SaaS platform that allows ANY ORGANIZATION (businesses, educational institutions, e-commerce, etc.) to connect their Meta Business accounts (Facebook Pages & Instagram) to AI-powered chatbots. The platform acts as a CONNECTION MANAGER, KNOWLEDGE BASE REPOSITORY, and WEBHOOK ROUTER.

**Business Model:**
1. Organizations register on the platform
2. Connect their Meta Business accounts via OAuth
3. Add Knowledge Bases (separate for chatbot messages and comments)
4. Request workflows/automations with optional add-ons
5. Seekers AI team receives notifications, builds n8n workflows, assigns webhooks
6. Platform routes all incoming messages/comments to client-specific workflows

**Key Architecture:**
- Meta App receives webhooks from Facebook/Instagram
- SaaS Platform manages: client connections, knowledge bases, workflow requests
- Admin Panel (Seekers) manages: workflow setup, webhook assignment, client monitoring
- n8n Server (Seekers AI) hosts individual workflows per client
- AI Processing happens in n8n workflows (using client's knowledge base)

**Company:** Seekers AI (Cairo, Egypt)
**Product:** Seekers AI Platform (Multi-Industry)
**Target:** Any organization requiring Meta chatbot automation

---

## TECHNICAL STACK

### Backend
- **Runtime:** Node.js 20+ with TypeScript
- **Framework:** Express.js or Fastify
- **Database:** PostgreSQL 15+
- **Cache/Queue:** Redis + Bull for job queues
- **ORM:** node-postgres (pg) with SQL queries
- **Auth:** JWT with bcrypt password hashing
- **Security:** helmet, cors, express-rate-limit

### Infrastructure
- **Hosting:** Docker containers (can deploy to Railway/Render/AWS)
- **Database:** PostgreSQL (Supabase or self-hosted)
- **File Storage:** Local initially, S3 for production
- **Logging:** Winston
- **Monitoring:** Optional (Sentry for errors)

### APIs & Integrations
- **Meta Graph API v18.0+**
- **n8n Webhooks** (custom Seekers AI instance)
- **OpenAI API** (optional, for testing - main AI in n8n)

CREATE OR REPLACE FUNCTION update_updated_at_column()
RETURNS TRIGGER AS $$
BEGIN
  NEW.updated_at = NOW();
  RETURN NEW;
END;
$$ language 'plpgsql';

### 1. Meta Service (`services/meta.service.ts`)
```typescript
import axios from 'axios';
import crypto from 'crypto';

export class MetaService {
  private readonly APP_ID = process.env.META_APP_ID!;
  private readonly APP_SECRET = process.env.META_APP_SECRET!;
  private readonly REDIRECT_URI = process.env.META_REDIRECT_URI!;
  private readonly API_VERSION = process.env.META_API_VERSION || 'v18.0';
  
  private readonly REQUIRED_SCOPES = [
    'pages_show_list',
    'pages_read_engagement',
    'pages_manage_metadata',
    'pages_manage_engagement',
    'instagram_basic',
    'instagram_manage_messages',
    'instagram_manage_comments',
    'public_profile',
    'email'
  ];

  /**
   * Generate Meta OAuth URL with CSRF protection
   */
  generateAuthUrl(state: string): string {
    const params = new URLSearchParams({
      client_id: this.APP_ID,
      redirect_uri: this.REDIRECT_URI,
      state: state,
      scope: this.REQUIRED_SCOPES.join(','),
      response_type: 'code'
    });
    
    return `https://www.facebook.com/${this.API_VERSION}/dialog/oauth?${params.toString()}`;
  }

  /**
   * Exchange authorization code for access token
   */
  async exchangeCodeForToken(code: string): Promise<{
    access_token: string;
    token_type: string;
    expires_in: number;
  }> {
    const response = await axios.get(
      `https://graph.facebook.com/${this.API_VERSION}/oauth/access_token`,
      {
        params: {
          client_id: this.APP_ID,
          client_secret: this.APP_SECRET,
          redirect_uri: this.REDIRECT_URI,
          code: code
        }
      }
    );
    
    return response.data;
  }

  /**
   * Exchange short-lived token for long-lived token (60 days)
   */
  async getLongLivedToken(shortToken: string): Promise<{
    access_token: string;
    token_type: string;
    expires_in: number;
  }> {
    const response = await axios.get(
      `https://graph.facebook.com/${this.API_VERSION}/oauth/access_token`,
      {
        params: {
          grant_type: 'fb_exchange_token',
          client_id: this.APP_ID,
          client_secret: this.APP_SECRET,
          fb_exchange_token: shortToken
        }
      }
    );
    
    return response.data;
  }

  /**
   * Get user's Facebook Pages with Instagram accounts
   */
  async getUserPages(userToken: string): Promise<any[]> {
    const response = await axios.get(
      `https://graph.facebook.com/${this.API_VERSION}/me/accounts`,
      {
        params: {
          access_token: userToken,
          fields: 'id,name,access_token,category,link,picture,instagram_business_account{id,username,profile_picture_url,followers_count}'
        }
      }
    );
    
    return response.data.data || [];
  }

  /**
   * Get user info from Meta
   */
  async getUserInfo(userToken: string): Promise<{
    id: string;
    name: string;
    email: string;
```

### 3. Token Service (`services/token.service.ts`)
```typescript
import crypto from 'crypto';
import { Pool } from 'pg';
import { logger } from '../config/logger';

const ALGORITHM = 'aes-256-gcm';
const ENCRYPTION_KEY = Buffer.from(process.env.ENCRYPTION_KEY!, 'hex');

export class TokenService {
  constructor(private db: Pool) {}

  /**
   * Encrypt access token before storing
   */
  encryptToken(token: string): string {
    const iv = crypto.randomBytes(16);
    const cipher = crypto.createCipheriv(ALGORITHM, ENCRYPTION_KEY, iv);
    
    let encrypted = cipher.update(token, 'utf8', 'hex');
    encrypted += cipher.final('hex');
    
    const authTag = cipher.getAuthTag();
    
    // Format: iv:authTag:encrypted
    return `${iv.toString('hex')}:${authTag.toString('hex')}:${encrypted}`;
  }

  /**
   * Decrypt access token when retrieving
   */
  decryptToken(encryptedData: string): string {
    const parts = encryptedData.split(':');
    if (parts.length !== 3) {
      throw new Error('Invalid encrypted token format');
    }

    const iv = Buffer.from(parts[0], 'hex');
    const authTag = Buffer.from(parts[1], 'hex');
    const encrypted = parts[2];
    
    const decipher = crypto.createDecipheriv(ALGORITHM, ENCRYPTION_KEY, iv);
    decipher.setAuthTag(authTag);
    
    let decrypted = decipher.update(encrypted, 'hex', 'utf8');
    decrypted += decipher.final('utf8');
    
    return decrypted;
  }

  /**
   * Store Meta user token
   */
  async storeUserToken(
    institutionId: string,
    userId: string,
    accessToken: string,
    expiresIn: number,
    metaUserId: string,
    metaUserName: string,
    scopes: string[]
  ): Promise<void> {
    const encrypted = this.encryptToken(accessToken);
    const expiresAt = new Date(Date.now() + expiresIn * 1000);

    await this.db.query(
      `INSERT INTO meta_tokens 
       (institution_id, user_id, access_token, token_type, expires_at, scopes, meta_user_id, meta_user_name)
       VALUES ($1, $2, $3, 'user_token', $4, $5, $6, $7)
       ON CONFLICT (institution_id, user_id) 
       DO UPDATE SET 
         access_token = EXCLUDED.access_token,
         expires_at = EXCLUDED.expires_at,
         scopes = EXCLUDED.scopes,
         updated_at = NOW()`,
      [institutionId, userId, encrypted, expiresAt, scopes, metaUserId, metaUserName]
    );
  }

  /**
   * Store Facebook page token
   */
  async storePageToken(
    institutionId: string,
    pageId: string,
    pageName: string,
    pageAccessToken: string,
    expiresIn: number | null,
    additionalData?: any
  ): Promise<string> {
    const encrypted = this.encryptToken(pageAccessToken);
    const expiresAt = expiresIn ? new Date(Date.now() + expiresIn * 1000) : null;

    const result = await this.db.query(
      `INSERT INTO facebook_pages 
       (institution_id, page_id, page_name, page_access_token, token_expires_at, page_category, page_link, page_picture_url)
       VALUES ($1, $2, $3, $4, $5, $6, $7, $8)
       ON CONFLICT (page_id) 
       DO UPDATE SET 
         page_access_token = EXCLUDED.page_access_token,
         token_expires_at = EXCLUDED.token_expires_at,
         page_name = EXCLUDED.page_name,
         updated_at = NOW()
       RETURNING id`,
      [
        institutionId,
        pageId,
        pageName,
        encrypted,
        expiresAt,
        additionalData?.category || null,
        additionalData?.link || null,
        additionalData?.picture?.data?.url || null
      ]
    );

    return result.rows[0].id;
  }

  /**
   * Get decrypted page token
   */
  async getPageToken(pageId: string): Promise<string | null> {
    const result = await this.db.query(
      `SELECT page_access_token FROM facebook_pages WHERE page_id = $1`,
      [pageId]
    );

    if (result.rows.length === 0) {
      return null;
    }

    return this.decryptToken(result.rows[0].page_access_token);
  }

  /**
   * Get decrypted user token
   */
  async getUserToken(institutionId: string): Promise<string | null> {
    const result = await this.db.query(
      `SELECT access_token FROM meta_tokens 
       WHERE institution_id = $1 AND token_type = 'user_token'
       ORDER BY created_at DESC LIMIT 1`,
      [institutionId]
    );

    if (result.rows.length === 0) {
      return null;
    }

    return this.decryptToken(result.rows[0].access_token);
  }

  /**
   * Check if tokens need refresh (within 7 days of expiry)
   */
  async getTokensNeedingRefresh(): Promise<Array<{
    institutionId: string;
    tokenId: string;
    expiresAt: Date;
  }>> {
    const result = await this.db.query(
      `SELECT id as token_id, institution_id, expires_at
       FROM meta_tokens
       WHERE expires_at < NOW() + INTERVAL '7 days'
         AND expires_at > NOW()
       ORDER BY expires_at ASC`
    );

    return result.rows;
  }
}
```

### 4. Webhook Processor Worker (`workers/webhookProcessor.ts`)
```typescript
import Queue from 'bull';
import { Pool } from 'pg';
import { WebhookRouterService } from '../services/webhookRouter.service';
import { logger } from '../config/logger';

const webhookQueue = new Queue('webhooks', {
  redis: {
    host: process.env.REDIS_HOST,
    port: parseInt(process.env.REDIS_PORT || '6379'),
    password: process.env.REDIS_PASSWORD
  },
  defaultJobOptions: {
    attempts: 3,
    backoff: {
      type: 'exponential',
      delay: 2000
    },
    removeOnComplete: 100,
    removeOnFail: 500
  }
});

interface WebhookJob {
  platform: 'facebook' | 'instagram';
  eventType: string;
  assetId: string; // page_id or instagram_account_id
  payload: any;
  receivedAt: string;
}

export async function initWebhookProcessor(db: Pool) {
  const routerService = new WebhookRouterService(db);

  webhookQueue.process(async (job) => {
    const { platform, eventType, assetId, payload } = job.data as WebhookJob;

    try {
      logger.info(`Processing ${platform} webhook for ${assetId}`);

      // Log webhook event
      await db.query(
        `INSERT INTO webhook_events 
         (platform, event_type, platform_id, payload, processed, created_at)
         VALUES ($1, $2, $3, $4, false, NOW())`,
        [platform, eventType, assetId, JSON.stringify(payload)]
      );

      // Route to n8n based on platform
      if (platform === 'facebook') {
        await routerService.routeFacebookWebhook(assetId, payload);
      } else if (platform === 'instagram') {
        await routerService.routeInstagramWebhook(assetId, payload);
      }

      // Mark as processed
      await db.query(
        `UPDATE webhook_events 
         SET processed = true, processed_at = NOW()
         WHERE platform_id = $1 AND event_type = $2 
           AND created_at > NOW() - INTERVAL '5 minutes'`,
        [assetId, eventType]
      );

      logger.info(`Successfully processed ${platform} webhook for ${assetId}`);

    } catch (error: any) {
      logger.error(`Failed to process webhook:`, error);

      // Log error
      await db.query(
        `UPDATE webhook_events 
         SET error_message = $1
         WHERE platform_id = $2 AND event_type = $3 
           AND created_at > NOW() - INTERVAL '5 minutes'`,
        [error.message, assetId, eventType]
      );

      throw error; // Bull will retry
    }
  });

  // Queue event listeners
  webhookQueue.on('completed', (job) => {
    logger.info(`Webhook job ${job.id} completed`);
  });

  webhookQueue.on('failed', (job, err) => {
    logger.error(`Webhook job ${job?.id} failed:`, err);
  });

  webhookQueue.on('stalled', (job) => {
    logger.warn(`Webhook job ${job.id} stalled`);
  });

  logger.info('Webhook processor initialized');

  return webhookQueue;
}

// Export function to add jobs to queue
export async function queueWebhook(data: WebhookJob) {
  await webhookQueue.add(data, {
    priority: 1
  });
}
```

```typescript
import { Router, Request, Response } from 'express';
import { MetaService } from '../services/meta.service';
import { queueWebhook } from '../workers/webhookProcessor';
import { logger } from '../config/logger';

const router = Router();
const metaService = new MetaService();

// Facebook webhook verification
router.get('/facebook', (req: Request, res: Response) => {
  const mode = req.query['hub.mode'];
  const token = req.query['hub.verify_token'];
  const challenge = req.query['hub.challenge'];

  if (mode === 'subscribe' && token === process.env.META_WEBHOOK_VERIFY_TOKEN) {
    logger.info('Facebook webhook verified');
    res.status(200).send(challenge);
  } else {
    logger.warn('Facebook webhook verification failed');
    res.sendStatus(403);
  }
});

// Facebook webhook receiver
router.post('/facebook', async (req: Request, res: Response) => {
  try {
    const signature = req.headers['x-hub-signature-256'] as string;
    const payload = JSON.stringify(req.body);

    // Validate signature
    if (!metaService.validateWebhookSignature(signature, payload)) {
      logger.warn('Invalid Facebook webhook signature');
      return res.sendStatus(403);
    }

    // Quick response to Meta
    res.sendStatus(200);

    // Process webhook asynchronously
    const body = req.body;

    if (body.object === 'page') {
      for (const entry of body.entry) {
        const pageId = entry.id;

        // Handle different event types
        if (entry.changes) {
          for (const change of entry.changes) {
            await queueWebhook({
              platform: 'facebook',
              eventType: change.field,
              assetId: pageId,
              payload: {
                field: change.field,
                value: change.value,
                entry: entry
              },
              receivedAt: new Date().toISOString()
            });
          }
        }

        // Handle messaging events
        if (entry.messaging) {
          for (const event of entry.messaging) {
            await queueWebhook({
              platform: 'facebook',
              eventType: 'messages',
              assetId: pageId,
              payload: {
                field: 'messages',
                value: event,
                entry: entry
              },
              receivedAt: new Date().toISOString()
            });
          }
        }
      }
    }

  } catch (error) {
    logger.error('Facebook webhook processing error:', error);
    // Still return 200 to Meta to avoid retries
    res.sendStatus(200);
```

## N8N WEBHOOK PROVISIONING MODEL

### Architecture Overview
The SaaS platform routes incoming Meta webhooks to n8n workflows. The routing model differs by subscription tier:

### Free Tier - Shared Base Webhook
All free-tier organizations share a **single base n8n webhook URL**. The n8n workflow behind this URL contains:
1. **AI Agent node** - Processes incoming messages/comments
2. **Router node** - Reads `organizationId` from payload to determine which knowledge base to use
3. **Knowledge Base lookup** - Fetches the client's KB content via the SaaS API
4. **Social Media API nodes** - Sends responses back via Meta Graph API

**How it works:**
```
Meta Webhook → SaaS Platform → Base n8n Webhook URL
                                    ↓
                              Router (by orgId)
                                    ↓
                         Client Knowledge Base lookup
                                    ↓
                              AI Agent processes
                                    ↓
                         Reply via Meta Graph API
```

**Payload sent to base webhook:**
```json
{
  "organizationId": "uuid",
  "organizationName": "Client Corp",
  "planType": "free",
  "platform": "facebook",
  "eventType": "messages",
  "pageId": "123456",
  "knowledgeBaseId": "uuid",
  "event": { /* original Meta webhook payload */ },
  "timestamp": "2024-01-15T10:30:00Z"
}
```

### Paid Tiers - Dedicated Webhooks
Paid clients (starter/professional/enterprise) get **dedicated n8n workflows** with unique webhook URLs:
1. Seekers admin creates a custom n8n workflow for the client
2. Admin enters the webhook URL in the SaaS admin panel (n8n_workflows table)
3. Admin links the workflow to specific Facebook pages/Instagram accounts (workflow_connections table)
4. Platform routes webhooks directly to the client's dedicated n8n workflow

### Admin Provisioning Flow
```
1. Client requests workflow → workflow_requests table (status: pending)
2. Admin reviews request → assigns to admin team member
3. Admin builds n8n workflow on Seekers n8n server
4. Admin enters webhook URL in SaaS admin panel
5. Admin links workflow to client's pages/accounts
6. Admin marks request as completed → client notified
```

### Environment Variables
```bash

# Base URL for dedicated webhooks
```

```typescript
// Middleware: Check usage limits before processing
async function checkUsageLimits(req, res, next) {
  const org = req.organization;
  const subscription = await getSubscription(org.id);
  const plan = await getPlan(subscription.plan_id);

  if (subscription.conversations_used >= plan.max_conversations_per_month) {
    return res.status(429).json({
      error: 'Monthly conversation limit reached',
      limit: plan.max_conversations_per_month,
      used: subscription.conversations_used,
      upgradeUrl: '/settings/billing'
    });
  }

  next();
}
```

## WORKERS SPECIFICATION

### 1. Token Auto-Refresh Worker (`workers/tokenRefresher.ts`)
```
Schedule: Every 6 hours
Logic:
  1. Query meta_tokens WHERE expires_at < NOW() + INTERVAL '7 days' AND expires_at > NOW()
  2. For each expiring token:
     a. Decrypt current token
     b. Call Meta Graph API to exchange for new long-lived token
     c. Encrypt and store new token
     d. Update expires_at
  3. If refresh fails:
     a. Log error
     b. Send notification to admin + client
     c. Mark token as needs_attention
```

### 2. Notification Dispatcher Worker (`workers/notificationWorker.ts`)
```
Trigger: Bull queue 'notifications'
Logic:
  1. Receive notification job from queue
  2. Store in notifications table
  3. Push via Socket.IO if user is online
  4. Send email if high priority or user offline
```

### 3. Usage Aggregation Worker (`workers/usageAggregator.ts`)
```
Schedule: Every hour
Logic:
  1. Count conversations per org for current period
  2. Update subscriptions.conversations_used
  3. Check against plan limits
  4. Send warning notifications at 80% and 100%
```
